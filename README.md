# Project-UAS-SSD_Paulus-Adidama-Prasada-124490040
“Project ini mengumpulkan sekitar 300 foto langit dari Asrama TB3 ITERA untuk memprediksi potensi hujan lebat. Foto diambil dalam berbagai kondisi cuaca seperti cerah, berawan, mendung, dan hujan. Data ini digunakan untuk melatih model klasifikasi gambar agar mampu mengenali tanda-tanda hujan.”
